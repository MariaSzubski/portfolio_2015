Insight

An Interactive Standardized Testing Solution

About This Project

Elevator Pitch

When science classes use fact memorization to prepare for standardized tests, students lose interest and fail to see the connection to the real word.


This project rethinks standardized testing though tablet and web based approach. It allows students to demonstrate conceptual knowledge and creative problem solving in an engaging, exploratory format.

What is a Capstone Project?

inSight is my senior capstone project, collaborating with Matt Denzer. This year-long project consisted of research, design, user testing and documentation.

This year-long project consisted of research, design, user testing and documentation. Goal: to create a tablet application that could be a new media format for standardized testing. The goal is to truely test student's knowledge and understanding of concepts, rather than memorizing facts. Each 'question' is actually an interactive piece that users go through to demonstrate skills and explain their thought process. We wanted to incorporate game elements and fun graphics to make it more engaging. Skills: Research User Testing UI Design HTML jQuery PhoneGap The final demo show two types of questions. In the first, users watch a video about a scientific concept and then explain what is happening in the video In the second, users are asked to demonstate their ability to conduct and evaluate an experiment ---- Our initial research included local museums, current standardized test formats, educational iPad apps, educational books, interviews with local educators. ---- In the first round of design, we chose to focus on science. We chose two questions and then began building the interface. In additoin to the questions, we created wireframes for the set up and question browsing screens. ---- We then tested these designs on some local students. We chose to test using a paper prototype of the design. We tested 7(?) students and asked them to go though each task. We wrote out a formal script for the user test. We then took this feedback and re-worked some of the designs. Details about our user test and results can be found on pages 23-28 in the process PDF (linked below). -- The final demo was built in HTML and wrapped in PhoneGap to get it to be functional on an iPad. Some elements of the demo are interactive, while some are motion test that demo the intended interaction. A working demo of inSight was displayed at the University of Cincinnati DAAPWorks 2012 from June 5th to June 8th
