About This Project

Guitar Practice Log is a web app that tracks skill progress and displays song details. The 1 Minute Chord Changes section logs finger exercises and charts progress over time. The Song List section displays song chords and strumming patterns.

Guitar Practice Log is an ongoing personal project that demonstrates project planning, design, and JavaScript skills. View this project’s progress on GitHub and the GitHub Wiki.

Project Sections

One Minute Chord Changes

This finger exercise builds muscle memory and increases chord change speed. The goal is to switch between two chords as fast as possible for sixty seconds.

This section includes a JavaScript timer with an animated ring. Users save the number of changes changes for each chord group and view trendlines that chart progress over time.

Song Notes

Users can view and add notes about songs they’ve learned, including chords and strumming patterns. This section is intended to be a quick reference while playing rather than true guitar tabs. The song list and detail views are built and transitioned using JavaScript and JSON.
